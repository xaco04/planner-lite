# Planner Lite

Planner Lite es una aplicación de planificación de tareas simple, inspirada en herramientas como LiquidPlanner, cuyo objetivo principal es aprender y demostrar prácticas de DevOps (Docker, Kubernetes, CI/CD).

## Objetivos
- Backend REST con Node.js y MongoDB
- Frontend web simple
- Contenerización con Docker
- Orquestación con Kubernetes
- CI/CD con GitHub Actions
